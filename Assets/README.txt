SETUP DEFAULT
1. Set scene called 'Main Menu' as your first build scene.
2. Change url on Button 'Game Jam Logo' (Main Menu Prefab> Canvas> MainMenu> Game Jam Logo)
3. Add the Pause Menu prefab to your own game scene.


CUSTOM
You don't need to use the assets as shown.
Add, remove or edit the main menu.
Remove or add items to the settings.
You don't need to add the pause menu to your game scene.

Change the prefabs or code to your liking!
